def is_palindrome(s: str) -> bool:

### 3ï¸âƒ£ Count Character Frequency

```python
def char_frequency(s: str) -> Dict[str, int]:
    freq = {}
    for ch in s:
        freq[ch] = freq.get(ch, 0) + 1
    return freq
```

---

### 4ï¸âƒ£ Check if Two Strings are Anagrams

```python
def are_anagrams(s1: str, s2: str) -> bool:
    if len(s1) != len(s2):
        return False

    freq = {}
    for ch in s1:
        freq[ch] = freq.get(ch, 0) + 1

    for ch in s2:
        if ch not in freq or freq[ch] == 0:
            return False
        freq[ch] -= 1

    return True
```

ðŸ‘‰ Pythonic shortcut:

```python
def are_anagrams_pythonic(s1: str, s2: str) -> bool:
    return sorted(s1) == sorted(s2)
```

### 5ï¸âƒ£ First Non-Repeating Character

```python
def first_non_repeating_char(s: str) -> Optional[str]:
    freq = {}

    for ch in s:
        freq[ch] = freq.get(ch, 0) + 1

    for ch in s:
        if freq[ch] == 1:
            return ch

    return None
```
